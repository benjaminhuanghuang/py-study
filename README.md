Python Playground
=====================================
Transforming Code into Beautiful, Idiomatic Python

## Setup
```
  $ python3 -m venv venv3

  $ . venv3/bin/activate
  or
  > .\venv3\Scripts\activate.bat                # windows
  > .\venv3\Scripts\Activate.ps1                # windows

  (venv3)$ pip3 install -r requirements.txt


  (venv3)$ python3 -m notebook
  ...
  (venv3)$ deactivate
```
